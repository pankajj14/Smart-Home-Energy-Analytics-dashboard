# Smart Home Energy Analytics Dashboard

## 📊 Project Overview
This project is a Power BI dashboard designed to analyze and visualize energy consumption in a smart home.

## 🚀 Features
- Real-time energy usage insights
- Interactive filters
- KPI-based analysis
- Trend visualization

## 🛠 Tools Used
- Power BI
- Excel / CSV

## 🎯 Purpose
To understand energy usage and help in making better energy decisions.
